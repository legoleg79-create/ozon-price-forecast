# Прогноз дохода Ozon

Статический калькулятор для прогноза прибыли при изменении цены товара на Ozon.

## Что считает

- комиссию Ozon;
- эквайринг;
- комиссию за заказ;
- доставку по тарифам Ozon с 01.05.2026;
- сценарий логистики "от" и "до";
- прибыль и ROI по диапазону цен.

Тарифы доставки встроены в `data/tariffs.js` и подготовлены из файла `logistika-fbo-fbs-01052026_1777018200.xlsx`.
